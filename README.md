# fdp-cli
##一款基于nodeJS快速创建react+typesrcript的构建wepapp项目的手脚架
###简介
#####  1.内置兼容IE8+浏览器。
#####  2.支持多语言i18next。
#####  3.支持多环境打包。
#####  4.搭载redux,redux,react-router,react-redux,redux-saga。
#####  5.集成了antd UI 3.X版本组件库,less定制主题,按需加载。
#####  6.集成了typerscript 类型语法。
### 使用
####快速创建语法
```npx @frid/fdp-cli init projectName```
